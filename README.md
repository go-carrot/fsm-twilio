# FSM Twilio

FSM CLI is a Twilio SMS/MMS target for [go-carrot/fsm](https://github.com/go-carrot/fsm).

## Environment Variables

When using this target, you must set three environment variables:

```
PORT=""
TWILIO_ACCOUNT_SID=""
TWILIO_AUTH_TOKEN=""
TWILIO_NUMBER="" # +15558675309
```
